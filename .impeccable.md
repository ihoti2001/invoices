## Design Context

### Users
Solo web designer (Ilir Hoti) — private admin tool used daily for invoicing, client management, and financial reporting. Single user, no multi-tenant needs. Used in a professional work context during business hours.

### Brand Personality
Precise · Composed · Unhurried

A web designer's tool that reflects their own taste — not a generic SaaS product. Should feel like something the user chose, not something that was inflicted on them.

### Aesthetic Direction
**Sleek monochromatic light** — one strong teal-slate accent colour, everything else in a tinted off-white palette. Feels like a Swiss-influenced design studio tool. Confident and quiet at the same time.

- **Theme**: Light (off-white/cream — `oklch(96.5% 0.009 210)`)
- **Accent**: Teal-slate — `oklch(42% 0.11 200)`
- **Sidebar**: Deep teal-navy — `oklch(17% 0.026 215)`, active nav item uses inverted white chip
- **Display font**: Bricolage Grotesque (variable grotesque with character — not a reflex pick)
- **Body font**: Hanken Grotesk (refined, professional, readable)
- **Anti-references**: No generic SaaS blue, no glassmorphism, no AI-slop card soup with gray-50 headers

### Design Principles
1. **Typographic hierarchy over chrome** — let font size, weight, and spacing do the work. Avoid UI decoration that adds noise.
2. **One accent, used sparingly** — teal-slate is the only colour with saturation. Everything else is tinted neutral.
3. **No card soup** — sections are distinguished by spacing and type, not identical bordered boxes with gray sub-headers.
4. **Inverted active state on sidebar** — active nav item gets a white chip on the dark background. High contrast, memorable, distinctive.
5. **Stats as typography** — key numbers displayed large and direct, without wrapping them in icon-row lists or KPI cards.
